系统要求
===============
 + php5.6+
 + MySQL5.7+

安装步骤：
===============
 + 第一步：跟目录执行 Composer install 或 Composer update
    + 注：Composer不了解请自行搜索了解，推荐查看官方[完全开发手册](https://www.kancloud.cn/manual/thinkphp5_1/353948)
 + 第二步：执行SQL文件（根目录sql.sql文件）
